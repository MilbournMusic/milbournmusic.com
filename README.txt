Milbourn Music — Les Mis Quiz (Web Version, no image)

Files:
- index.html
- style.css
- script.js

How to run locally:
- Double-click index.html, or use a simple server (python -m http.server).

Deploy:
- GitHub Pages: push these files to a public repo, enable Pages (Branch: main, / (root)).
- Then set your custom domain in the repo Settings → Pages, and add DNS records at your registrar.
